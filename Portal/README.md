# Portal

**Purpose:** Drop box for external items and extraction point for docker/workspace content.

## Usage

### Incoming (From Outside)
Place files here that are sent from external sources — uploads, shared documents, data dumps, etc.

### Outgoing (From Docker/Workspace)
Move files here that need to be pulled out of the docker environment — exports, deliverables, reports, etc.
