# content(lab): Add public-apis usefulness review

Reviewed ~400 APIs from https://github.com/public-apis/public-apis

Identified:
- Top 10 Most Useful: GitHub API, Auth0, VirusTotal, Nager.Date, Storj, Mailjet, Google Safe Browsing, IPstack, CoinGecko, Pantry
- 10 More Hidden Gems: Mailboxlayer, Numverify, Marketstack, URLScan.io, AbuseIPDB, Web3 Storage, Metropolitan Museum, Pinata, Gitter, Open Library
- 9 Gaps (missing industry standards): Stripe, Twilio, AWS SDK, SendGrid, Supabase, Clerk, Vercel API, Firebase, Cloudflare

Security warnings flagged:
- AnonFiles, BayFiles, 0x0.st (anonymous file hosting — abuse vectors)
- Public CORS proxies (security risk)

5-tier classification system:
- 🔴 Critical (infrastructure)
- 🟠 High (production-ready)
- 🟡 Medium (niche/prototyping)
- 🟢 Low (novelty)
- ⚪ Avoid (security/deprecated)

Pareto observation: ~20% of APIs provide 80% of real-world value.
Crypto APIs overrepresented (50+ redundant exchanges).
