# Six Themes, One Website — An Experiment in Perspective

**Type:** Experiment
**Date:** 2026-02-27
**For:** promptengines.com

---

Most websites pick a design and stick with it. We have six.

When you visit promptengines.com, you can cycle through six completely different visual identities for the same site — each one built on the same content, but communicating something different about what PromptEngines is:

- **Copper / Topographic** — the default. Warm, tactile, like a map of something being explored.
- **Green Terminal / Bloomberg** — raw data, command-line energy, for when you want to see the machinery.
- **Military / Defense Contractor** — serious, precise, mission-oriented.
- **Gallery White** — editorial, clean, the art museum version.
- **Matrix Hacker** — you know what this is.
- **Pixel City Isometric** — playful, generative, worlds within worlds.

This wasn't just aesthetic experimentation. We wanted to understand something: does the frame change how people perceive the work? The answer, so far, is yes — dramatically.

The same product descriptions read completely differently depending on which lens you're looking through. The Terminal theme makes everything feel like infrastructure. The Gallery theme makes it feel like craft. Neither is wrong — they're just different truths about the same thing.

We use a "Switch Perspective" button that cycles through all six. It's a small interaction, but it makes the point: the work doesn't change. The story you tell about it does.

We're still running this experiment. If you find one that clicks for you, that tells us something.
