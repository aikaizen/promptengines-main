Tools

🚀 Quick Start: Generate Your First Lesson

Get API keys

Fireworks AI for images, ElevenLabs for audio (or Amazon Polly for budget option)

Configure 
cd tools/
cp config.json.template config.json
# Add your API keys
Generate images 
node image-gen/batch-generate.js --lesson quest-001-letter-a
Generate audio 
node audio-gen/batch-audio.js --lesson quest-001-letter-a
Review outputs

Check assets/images/ and assets/audio/ — iterate on prompts if needed

Available Tools

🔧 batch-generate.js

✓ Ready Node.js Fireworks AI

Batch image generation for lesson assets. Reads asset manifests and generates all images with consistent cartoon style.

Usage

# Generate all images for a lesson
node image-gen/batch-generate.js --lesson quest-001-letter-a

# Force regeneration (overwrite existing)
node image-gen/batch-generate.js --lesson quest-001-letter-a --force

# Generate specific category only
node image-gen/batch-generate.js --lesson quest-001-letter-a --category characters

Features

  • Automatic style template application (consistent cartoon look)
  • Organized output by lesson and category
  • Generation report with success/failure tracking
  • Resume support (skips existing files)
  • Transparent PNG backgrounds for characters/objects

🔊 batch-audio.js

✓ Ready Node.js ElevenLabs / Polly

Batch audio generation for lesson narration and feedback. Supports ElevenLabs (high quality) or Amazon Polly (budget).

Usage

# Generate with ElevenLabs (recommended for MVP)
node audio-gen/batch-audio.js --lesson quest-001-letter-a --provider elevenlabs

# Generate with Polly (cheaper for scale)
node audio-gen/batch-audio.js --lesson quest-001-letter-a --provider polly

# Force regeneration
node audio-gen/batch-audio.js --lesson quest-001-letter-a --force

Features

  • Multi-provider support (ElevenLabs, Amazon Polly)
  • SSML support for Polly (pauses, emphasis, intonation)
  • Emotion/tone tags in manifest (cheerful, encouraging, calm)
  • MP3 output, 64-128kbps, <10MB per lesson
  • Generation report tracking success/failure

Cost Comparison

  • ElevenLabs: ~$2.05 per lesson (natural quality)
  • Amazon Polly: ~$0.13 per lesson (robotic but functional)

Asset Manifest Format

All tools read from lessons/{lesson-id}/assets-manifest.json:

{
  "lessonId": "quest-001-letter-a",
  "title": "Letter A Adventure",
  "duration": "6 minutes",
  "images": {
    "characters": [
      {
        "id": "tutor-happy",
        "prompt": "Friendly cartoon tutor character, smiling...",
        "size": "512x512"
      }
    ],
    "objects": [...],
    "letters": [...],
    "backgrounds": [...],
    "ui": [...]
  },
  "audio": {
    "narration": [
      {
        "id": "intro-welcome",
        "text": "Welcome to the Letter A Adventure!",
        "emotion": "cheerful",
        "pace": "normal"
      }
    ],
    "feedback": [...]
  }
}

Style Guidelines

Visual Style

2D cartoon, friendly and warm, simple shapes, bold outlines, flat colors, no gradients, educational style, high contrast, cheerful, transparent backgrounds

Audio Style

Clear, warm, encouraging, age-appropriate pacing, specific praise over generic "good job", emotion-matched to context

File Formats

Images: PNG (characters/objects with transparency), JPG (backgrounds). Audio: MP3, 64-128kbps. Max 200MB per lesson.

Naming Convention

kebab-case, descriptive, category-prefixed: tutor-happy.png, apple.png, intro-welcome.mp3
