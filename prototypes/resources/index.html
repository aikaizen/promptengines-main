01 — Essentials

Top 10 APIs

From ~400 public APIs reviewed, these are the critical infrastructure pieces. If we had to start with only 10 external services, these would be it.

GitHub API
#1

The foundation. Repos, issues, Actions, PRs—everything flows through here.

Critical DevOps
Auth0
#2

Production-grade authentication. Don't build auth yourself.

Critical Identity
VirusTotal
#3

File and URL security scanning. Essential for any app accepting uploads.

High Security

#4

Public holidays for 90+ countries. Surprisingly essential for scheduling logic.

High Calendar

#5

Decentralized cloud storage. 1TB free, S3-compatible, privacy-first.

Critical Storage

Critical Email

High Security

High Finance

High Storage

Gaps We Want to Fill

Notable APIs that should be on public-apis but aren't. These are industry standards we use despite the list's omissions.

API Category Why It Matters Status
Stripe Payments Industry standard for payment processing. Missing is a major gap. Critical Gap
Twilio Communication SMS, voice, email. The standard for programmable communications. Critical Gap
AWS SDK Cloud S3, Lambda, DynamoDB—the backbone of modern infrastructure. Critical Gap
SendGrid Email Enterprise email delivery. More robust than most listed alternatives. High Priority
Supabase Database Open source Firebase alternative. PostgreSQL + realtime. High Priority
Clerk Auth Modern auth with React components. Developer experience leader. High Priority
Vercel API Hosting Edge deployment, serverless functions. Our default host. High Priority
Firebase Backend Realtime database, auth, hosting. Google's BaaS platform. Medium Priority
Cloudflare Infrastructure CDN, Workers, D1 database. Edge computing leader. High Priority
