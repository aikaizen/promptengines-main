# Anti-Slop Writing Skill

Prevents low-signal, hype-driven, or generic AI-generated prose. Enforces Lab Notes editorial standards.

## Description

Writing quality enforcement tool that detects and rejects "slop" patterns common in AI-generated content. Promotes concrete, falsifiable, actionable prose.

## Usage

### As a Shell Script
```bash
source anti-slop.sh
```

### As an Editorial Checklist
Review content against the patterns listed in SKILL.md before publishing.

## Slop Patterns Detected

1. **Denial of Commerce** - Defining by negation
2. **Symbiosis Sermons** - Abstraction without anchoring
3. **Acceleration Invocations** - Timeless claims without quantification
4. **Vibe Adjective Stacks** - Marketing adjectives
5. **Hedging Preambles** - Wasted opening sentences
6. **Agentic Self-Reference** - Unnecessary ontology mentions

## Operational Rules

1. First sentence must contain noun + verb advancing understanding
2. No paragraph starts with "This is..." or "We are..."
3. All claims must be falsifiable or marked as opinion
4. Adjectives require justification
5. Abstract nouns must be replaced with concrete processes
